CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • Selectors determine whether an #id, .class, or section is affected

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git pull origin branch-name: pulls all files from branch
  • git push origin branch-name: pushes all files to branch
  • git config --global user.name "[firstname lastname]": assigns user email
  • git config --global user.email "[valid-email]": associated email address

JavaScript

  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
  • A group of data is held in an array.
  • A index i.e. index[0] is a unique number used to identify the location of a data item in an array.
