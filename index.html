Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git pull origin branch-name: pulls all files from branch
  • git push origin branch-name: pushes all files to branch
  • git config --global user.name "[firstname lastname]": assigns user email
  • git config --global user.email "[valid-email]": associated email address
